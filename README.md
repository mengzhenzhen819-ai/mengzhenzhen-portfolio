# Mengzhenzhen Portfolio

这是我的个人作品集网站项目。

## 项目状态

当前网站还在持续修改中，先上传到 GitHub 作为版本备份。

## 技术栈

- React
- Vite
- Tailwind CSS
- TypeScript
- Framer Motion
- Lucide React

## 主要目录

- `src/`：网站主要代码
- `src/sections/`：首页、关于我、项目、联系我等页面模块
- `src/components/`：通用组件
- `public/images/`：网站图片素材
- `package.json`：项目依赖配置

## 后续修改原则

后续让 AI 修改时，请基于现有代码继续修改，不要重新生成整个网站。

修改前需要先说明：
1. 会修改哪些文件
2. 为什么修改这些文件
3. 是否会影响其他页面
