这是当前完成版个人作品集网站的工程版源码包。

本包保留了之前源码包里的工程结构：

- `components.json`
- `eslint.config.js`
- `index.html`
- `package-lock.json`
- `package.json`
- `postcss.config.js`
- `public/`
- `src/`
- `tailwind.config.js`
- `tsconfig.app.json`
- `tsconfig.json`
- `tsconfig.node.json`
- `vercel.json`
- `vite.config.ts`

同时，本包已经放入当前完成版网页所需的静态页面和资源：

- `project-*.html`
- `assets/`
- `images/`

如果你是直接替换 GitHub 仓库文件，可以把本文件夹内所有内容放到仓库根目录。
